# REST-API-MongoDB
This is an ASP.NET Core API with MongoDB Driver functions.

* MongoDB with .NET Driver
* ASP.NET Core 5
* Swagger
* User Secrets

Nuget Packages:
* MongoDB Driver
* Swashbuckle.AspNetCore
